# **App Name**: BudgetWise

## Core Features:

- Income Logging: Log income transactions with specific categories (e.g., salary, investments).
- Expense Logging: Log expense transactions with categories (e.g., groceries, rent, utilities).
- Spending Breakdown: Visually display spending breakdowns by category, either as charts or lists.
- Budget Setting: Set monthly budget limits for each category to track overspending.
- Financial Summary: Display a quick financial summary including total income, total expenses, and net balance.
- AI-Powered Budget Tips: Provide AI-driven personalized recommendations for users to save more and manage expenses. This tool analyzes the budget and expenses, and will incorporate the suggestions in its summary of expenses.

## Style Guidelines:

- Primary color: Professional blue (#3498DB) for a sense of trust and stability.
- Background color: Light gray (#ECF0F1) for a clean and intuitive interface.
- Accent color: Green (#2ECC71) for income, and Red (#E74C3C) for expenses to visually differentiate positive and negative cash flow. The green and red should contrast to be distinct.
- Body and headline font: 'PT Sans', a humanist sans-serif with a modern, yet warm feel.
- Use clean and modern icons for each transaction category to enhance usability.
- Maintain a clean and well-organized layout for intuitive navigation.
- Use subtle animations, like transitions in charts.
